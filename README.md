# نظام كشف إشارات RF التعليمي

## ⚠️ تحذيرات أمنية مهمة

### هذا نظام تعليمي للتدريب فقط على:
- مفاهيم استقبال إشارات RF
- تحليل الطيف الترددي
- محاكاة أنظمة الاستشعار

### ❌ **لا يمكن استخدامه لـ:**
- كشف الدرون أو الطائرات
- المراقبة أو التجسس
- أي استخدام عسكري أو أمني
